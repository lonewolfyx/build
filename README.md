# build

A lightweight CLI tool to simplify frontend builds and artifact packaging. It acts as a shortcut for `pnpm build` / `npm run build` and automatically compresses the `dist` directory into an archive for easy delivery.

## Install

```bash
npm install @lonewolfyx/build -g
```

## Usage

```bash
build
```
